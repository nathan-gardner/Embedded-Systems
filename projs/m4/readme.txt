# m4_app

m4_app is the solution for the first part of Milestone 4 that
requires the implementation of Milestone 3 using ESOS.

# barrier

barrier is the solution for the second part of Milestone 4
that requires the implementation of one of four basic synchronization
semaphore patterns (multiplex, barrier, reusable barrier, queue)
from Allen B. Downey's "The Little Book of Semaphores" 2/e.
